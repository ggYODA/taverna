## 2. ГОТОВЫЕ SEO-ТЕГИ ДЛЯ index.html

### 2.1 `<title>`

```html
<title>Armellada Taverna — Greek Seafood Restaurant in Nea Makri | Ταβέρνα Νέα Μάκρη</title>
```

**Почему:** содержит бренд, основной тип заведения, локацию, два языка. Длина ~75 символов — в норме для Google.

### 2.2 `<meta name="description">`

```html
<meta name="description" content="Armellada Taverna — authentic Greek taverna in Nea Makri, Aeroporias 80. Fresh seafood, grilled octopus, local wines & Mediterranean cuisine. Ταβέρνα με φρέσκα θαλασσινά στη Νέα Μάκρη. Book a table: +30 699 442 4276.">
```

**Почему:** ~230 символов (в пределах допустимых 320). Включает: бренд, тип + локация, ключевые блюда, греческий текст, CTA с телефоном.

### 2.3 `<meta name="keywords">`

```html
<meta name="keywords" content="Armellada Taverna, Greek taverna Nea Makri, seafood restaurant Nea Makri, ταβέρνα Νέα Μάκρη, εστιατόριο θαλασσινών, Aeroporias 80, Greek cuisine, fish tavern Attica, ψαροταβέρνα, Mediterranean restaurant, best restaurant Nea Makri, grilled octopus, fresh fish Greece, Marathon restaurant, καλύτερο εστιατόριο Νέα Μάκρη">
```

> Примечание: Google официально не использует meta keywords для ранжирования, но Bing и другие поисковики могут учитывать. Не навредит.

## 4. ЗАГОЛОВКИ H1 / H2 (SEO-оптимизированные)

### H1 (один на страницу — уже в Hero)

**Текущий:** `Armellada Taverna`  
**Рекомендуемый (оставить как есть):** Бренд в H1 — это правильно. Подзаголовок `hero-subtitle` уже содержит ключевые слова.

Рекомендуется добавить скрытый `<span>` для поисковиков внутри hero:

```html
<h1 class="hero-title">
    <span class="hero-title-line">Armellada</span>
    <span class="hero-title-line hero-title-line--accent">Taverna</span>
    <span class="sr-only">— Greek Seafood Restaurant in Nea Makri, Greece | Ελληνική Ταβέρνα Νέα Μάκρη</span>
</h1>
```

## 5. ДОПОЛНИТЕЛЬНЫЕ РЕКОМЕНДАЦИИ

### 5.1 `<html lang>`
Сменить `lang="ru"` на `lang="en"` (основная аудитория — международная / англоязычная). Или использовать `lang="en"` с `hreflang`:

```html
<html lang="en">
```

### 5.2 Hreflang (если будут мультиязычные версии)
```html
<link rel="alternate" hreflang="en" href="https://armellada-taverna.gr/">
<link rel="alternate" hreflang="el" href="https://armellada-taverna.gr/el/">
<link rel="alternate" hreflang="ru" href="https://armellada-taverna.gr/ru/">
```

### 5.3 Файл sitemap.xml
```xml
<?xml version="1.0" encoding="UTF-8"?>
<urlset xmlns="http://www.sitemaps.org/schemas/sitemap/0.9">
  <url>
    <loc>https://armellada-taverna.gr/</loc>
    <lastmod>2026-03-02</lastmod>
    <changefreq>weekly</changefreq>
    <priority>1.0</priority>
  </url>
</urlset>
```

### 5.4 Файл robots.txt
```
User-agent: *
Allow: /
Sitemap: https://armellada-taverna.gr/sitemap.xml
```
